    创建数据库¶

    启动一个sqlsh会话，然后输入以下命令以创建并连接到数据库：

    1
2
    CREATE DATABASE testdb;
\c testdb;

    以下创建表的命令显示了如何将SQL语句分布在多行输入中。请注意更改提示：

\pset format unaligned

Output format is unaligned.

\pset fieldsep ,

Field separator is ",".

\pset tuples_only

Showing only tuples.

select second, first FROM my_table;

one,1
two,2
three,3
four,4
