    枚举类型

    枚举（enum）类型是由一个静态、值的有序集合构成的数据类型。它们等效于很多编程语言所支持的enum类型。枚举类型的一个示例可以是一周中的日期，或者一个数据的状态值集合。

    枚举类型的声明¶

    枚举类型可以使用CREATE TYPE命令创建，例如：

    1
    CREATE TYPE mood AS ENUM ('sad', 'ok', 'happy');

    一旦被创建，枚举类型可以像很多其他类型一样在表和函数定义中使用：

    1
2
3
4
5
6
7
8
9
    CREATE TYPE mood AS ENUM ('sad', 'ok', 'happy');
CREATE TABLE person (
    name text,
    current_mood mood
);

INSERT INTO person VALUES ('Moe', 'happy');

select * FROM person WHERE current_mood = 'happy';

    返回信息如下：

    1
2
3
     name | current_mood
------+--------------
 Moe  | happy

    排序¶

    一个枚举类型的值的排序是该类型被创建时所列出的值的顺序。枚举类型的所有标准的比较操作符以及相关聚集函数都被支持。例如：

    1
2
3
    INSERT INTO person VALUES ('Larry', 'sad');
INSERT INTO person VALUES ('Curly', 'ok');
select * FROM person WHERE current_mood > 'sad';

    返回信息如下：

    1
2
3
4
5
6
     name  | current_mood
-----------+------------------
 Moe   | happy
 Curly  | ok

select * FROM person WHERE current_mood > 'sad' ORDER BY current_mood;

    返回信息如下： name | current_mood

    1
2
3
4
5
6
    -----------+----------------
 Curly  | ok
 Moe   | happy
select name
FROM person
WHERE current_mood = (select MIN(current_mood) FROM person);

    返回信息如下：

    1
     Larry

    类型安全性¶

    每一种枚举数据类型都是独立的并且不能和其他枚举类型相比较。看这样一个示例：

    1
2
3
4
5
6
7
8
9
    CREATE TYPE happiness AS ENUM ('happy', 'very happy', 'ecstatic');
CREATE TABLE holidays (
    num_weeks integer,
    happiness happiness
);
INSERT INTO holidays(num_weeks,happiness) VALUES (4, 'happy');
INSERT INTO holidays(num_weeks,happiness) VALUES (6, 'very happy');
INSERT INTO holidays(num_weeks,happiness) VALUES (8, 'ecstatic');
INSERT INTO holidays(num_weeks,happiness) VALUES (2, 'sad');

    返回错误信息如下：

    1
2
3
    ERROR:  invalid 输入值 for enum happiness: "sad"
select person.name, holidays.num_weeks FROM person, holidays
  WHERE person.current_mood = holidays.happiness;

    返回错误信息如下：

    1
    ERROR:  operator does not exist: mood = happiness

    如果你确实需要做这样的事情，你可以写一个自定义的操作符或者在查询中加上显式造型：

    1
2
    select person.name, holidays.num_weeks FROM person, holidays
  WHERE person.current_mood::text = holidays.happiness::text;

    返回信息如下：

    1
2
3
     name | num_weeks
---------+--------------
 Moe  |         4

    枚举标签是大小写敏感的，因此'happy'与'HAPPY'是不同的。标签中的空格也是有意义的。

    尽管枚举类型的主要目的是用于值的静态集合，但也有方法在现有枚举类型中增加新值和重命名值（见ALTER TYPE）。不能从枚举类型中去除现有的值，也不能更改这些值的排序顺序，如果要那样做可以删除并且重建枚举类型。

    函数¶

    函数 描述
    enum_first() 返回输入枚举类型的第一个值。
    enum_last() 返回输入枚举类型的最后一个值。
    enum_range() 将输入枚举类型的所有值作为一个有序的数组返回。
    enum_range() 以一个数组返回在给定两个枚举值之间的范围。值必须来自相同的枚举类型。 如果第一个参数为空，其结果将从枚举类型的第一个值开始。如果第二参数为空，其结果将以枚举类型的最后一个值结束。

    假定一个枚举类型被创建为：

    1
    CREATE TYPE rainbow AS ENUM ('red', 'orange', 'yellow', 'green', 'blue', 'purple');

    enum_first()¶

    目的：返回输入枚举类型的第一个值。

    语法：

    输入值: anyenum
    返回值: anyenum

    例如：

    1
    select enum_first(null::rainbow);

    返回信息如下：

    1
    red

    enum_last()¶

    目的：返回输入枚举类型的最后一个值。

    语法：

    输入值: anyenum
    返回值: anyenum

    例如：

    1
    select enum_last(null::rainbow);

    返回信息如下：

    1
    purple

    enum_range()¶

    目的：将输入枚举类型的所有值作为一个有序的数组返回。

    语法：

    输入值: anyarray
    返回值: anyenum

    例如：

    1
    select enum_range(null::rainbow);

    返回信息如下：

    1
     {red,orange,yellow,green,blue,purple}

    enum_range()¶

    目的：以一个数组返回在给定两个枚举值之间的范围。值必须来自相同的枚举类型。 如果第一个参数为空，其结果将从枚举类型的第一个值开始。如果第二参数为空，其结果将以枚举类型的最后一个值结束。

    语法：

    输入值: anyenum, anyenum 返回值: anyarray

    例如：

    1
    select enum_range('orange'::rainbow, 'green'::rainbow);

    返回信息如下：

    1
    {orange,yellow,green}

    例如：

    1
    select enum_range(NULL, 'green'::rainbow);

    返回信息如下：

    1
    {red,orange,yellow,green}

    例如：

    1
    select enum_range('orange'::rainbow, NULL);

    返回信息如下：

    1
    {orange,yellow,green,blue,purple}
