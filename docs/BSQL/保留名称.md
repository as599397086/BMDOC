BSQL保留以下名称供内部使用。当使用这些名称时，即使它们被双引号引用，也会引发异常。

| 名称      | 描述           |
| --------- | -------------- |
| oid       | 系统列         |
| tableoid  | 系统列         |
| xmin      | 系统列         |
| cmin      | 系统列         |
| xmax      | 系统列         |
| cmax      | 系统列         |
| ctid      | 系统列         |
| bmctid    | 虚拟列         |
| 前缀为pg_ | 系统数据库对象 |
| 前缀为bm_ | 系统数据库对象 |
