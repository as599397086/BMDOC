    数学函数和操作符

    除非另有说明, 显示为可接受 numeric_type 的操作符对所有的 smallint、integer、bigint、numeric、real 和 double precision类型都可用。 显示为可接受 integral_type 的操作符对 smallint、integer 和 bigint类型是可用的。 除了特别说明之处，操作符的每种形式都返回与其参数相同的数据类型。 涉及多个参数数据类型的调用， 例如 integer + numeric，可通过使用这些列表中稍后出现的类型来解析。

    数学操作符¶

    操作符 描述
    + 加
    + 一元加(无操作)
    - 减
    - 否定
    * 乘
    / 除
    % 模（取余）
    ^ 指数
    [ /](#_
    [
    ! 阶乘
    !! 阶乘作为前缀操作符
    @ 绝对值
    & 按位与(AND)
    [ ](#_
    # 按位异或
    ~ 按位求反(NOT)
    << 按位左移
    >> 按位右移

    +¶

    目的：相加。

    例如：

    1
    select 2 + 3;

    返回信息如下：

    1
    5

    +¶

    目的：一元加(无操作)。

    例如：

    1
    select + 3.5;

    返回信息如下：

    1
    3.5

    -¶

    目的：减。

    例如：

    1
    select 2 - 3;

    返回信息如下：

    1
    -1

    -¶

    目的：否定。

    例如：

    1
    select - (-4);

    返回信息如下：

    1
    4

    *¶

    目的：乘。

    例如：

    1
    select 2 * 3;

    返回信息如下：

    1
    6

    /¶

    目的：除(对于整型，除法将结果截断为零)。

    例如：

    1
    select 5.0 / 2 ;

    返回信息如下：

    1
    2.5000000000000000

    例如：

    1
    select 5 / 2 ;

    返回信息如下：

    1
    2

    %¶

    目的：模（取余）; 适用于 smallint，integer，bigint 和 numeric。

    例如：

    1
    select 5 % 4;

    返回信息如下：

    1
    1

    ^¶

    目的：指数 (多次使用 ^ 将会从左到有关联) 。

    例如：

    1
    select 2 ^ 3;

    返回信息如下：

    1
    8

    例如：

    1
    select 2 ^ 3 ^ 3;

    返回信息如下：

    1
    512

    |/¶

    目的：平方根。

    例如：

    1
    select |/ 25.0 ;

    返回信息如下：

    1
    5

    ||/¶

    目的：立方根。

    例如：

    1
    select ||/ 64.0;

    返回信息如下： 

    1
    4

    !¶

    目的：阶乘(已弃用, 使用 factorial() 代替) 。

    例如：

    1
    select 5 !;

    返回信息如下：

    1
    120

    !!¶

    目的：阶乘作为前缀操作符(已弃用, 使用 factorial() 代替)。

    例如：

    1
    select !! 5;

    返回信息如下：

    1
    120

    @¶

    目的：绝对值。

    例如：

    1
    select @ -5.0 ;

    返回信息如下：

    1
    5

    &¶

    目的：按位与(AND)。

    例如：

    1
    select 91 & 15 ;

    返回信息如下：

    1
    11

    |¶

    目的：按位或(OR) 。

    例如：

    1
    select 32 | 3;

    返回信息如下：

    1
    35

    #¶

    目的：按位异或。

    例如：

    1
    select 17 # 5;

    返回信息如下：

    1
    20

    ~¶

    目的：按位求反(NOT)。

    例如：

    1
    select ~1 ;

    返回信息如下：

    1
    -2

    <<¶

    目的：按位左移。

    例如：

    1
    select 1 << 4;

    返回信息如下：

    1
    16

    >>¶

    目的：按位右移。

    例如：

    1
    select 8 >> 2;

    返回信息如下：

    1
    2

    函数 描述
    abs() 绝对值

    ceiling() 大于或等于参数的最接近的整数 (与 ceil 相同)
    degrees() 将弧度转换为角度
    div() y/x 的整数商
    exp() 指数 (e 的给定次方)
    factorial() 阶乘
    floor() 小于或等于参数的最接近整数
    ln() 自然对数
    log() 以10为底的对数
    log() 以指定值为底的对数
    mod() 余数
    pi() π的近似值
    power() a的b次幂
    radians() 将角度转换为弧度
    round() 四舍五入到最近的整数
    round() 四舍五入到指定的小数位
    scale() 小数点后的位数
    sign() 参数的符号
    sqrt() 平方根
    trunc() 截断整数
    trunc() 截断到指定小数位
    width_bucket() 返回包含等宽柱的柱状图中，操作数所在的柱的编号，范围从低到高。
    width_bucket() 返回一个柱号，这个柱是在给定数组中操作数将被分配的柱。

    width_bucket()¶

    目的：返回包含等宽柱的柱状图中，操作数所在的柱的编号，范围从低到高。 超出该范围的输入则返回0或计数+1。

    语法：

    输入值: double precision, double precision, double precision, integer 返回值: integer

    或者

    输入值: numeric, numeric, numeric, integer 返回值: integer

    例如：

    1
    select width_bucket(5.35, 0.024, 10.06, 5);

    返回信息如下：

    1
    3

    width_bucket()¶

    目的：返回一个柱号，这个柱是在给定数组中操作数将被分配的柱。 对于一个低于第一个下界的输入返回0。 操作数和数组元素可以是具有标准比较操作符的任何类型。 数组必须被排好序，最小的排在最前面，否则将会得到意想不到的结果。

    语法：

    输入值: anyelement, anyarray 返回值: integer

    例如：

    1
    select width_bucket(now(), array['yesterday', 'today', 'tomorrow']::timestamptz[]);

    返回信息如下：

    1
    2

    random()¶

    语法：

    返回值: double precision

    例如：

    setseed()¶

    目的：为后续的random()调用设置种子；参数必须在-1.0和1.0之间，包括边界值

    语法：

    输入值: double precision 返回值: void

    例如：

    1
    select setseed(0.12345);
